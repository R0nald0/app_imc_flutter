# Imc App

### Projeto/Aplicativo mobile para calcular Índice de massa corporal- IMC,feito em linguagem dart com framework flutter.
#### Funcionalidades

* listar histórico de calculados IMC

* calcular Imc

* Persistência de dados Local com Hive
